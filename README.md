# TUI-Raycaster
A raycaster for the terminal, made using the [Lanterna](https://github.com/mabe02/lanterna) library.

### Controls:

- 'w': Move forwards.
- 's': Move backwards.
- 'd': Move right.
- 'a': Move left.
- 'q' Rotate left.
- 'e' Rotate right.
- '+' Increase FOV.
- '-' Decrease FOV.
- 't' Increase view distance.
- 'g' Decrease view distance.
